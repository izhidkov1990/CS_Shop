# Agent Playbooks

These documents describe how different AI agents should operate in this repo.
Use the relevant agent guide when assigning tasks or asking for reviews.

Agents:
- Backend Agent: APIs, services, data models, auth flows.
- Frontend Agent: UI, client auth flow, integration with gateway.
- QA Agent: tests, regression checks, edge cases.
- DevOps Agent: environments, CI/CD, infrastructure, secrets.
- Security Agent: threat modeling, secrets, auth hardening.
